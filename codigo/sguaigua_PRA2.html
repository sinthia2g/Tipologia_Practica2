1.4.2 ComprobaciÃ³n de la normalidad y homogeneidad de la varianza.

Para la comprobaciÃ³n de que los valores que toman nuestras variables cuantitativas provienen de una poblaciÃ³n distribuida normalmente, utilizaremos la prueba de normalidad de AndersonDarling. AsÃ­, se comprueba que para que cada prueba se obtiene un p-valor superior al nivel de significaciÃ³n prefijado Î± = 0, 05. Si esto se cumple, entonces se considera que variable en cuestiÃ³n sigue una distribuciÃ³n normal. Se identifica que ninguna variable es normal.

alpha = 0.05
